    Microcontroller

    Option Pros Cons

    PIC18F47Q10-I/PT
    $1.65/each
    datasheet    		 - Familiar due to use in course

    - 128kB Memory

    - 36 IO pins

    - Fast 64MHz processing

    - Inexpensive    		 - Low power range

    - Could be limiting with amount of peripherals

    PIC24FJ256GA705-I/PT
    $2.13/each
    datasheet    		 - Supports 16-bit

    - 256kB Memory

    - 40 IO pins    		 - Unfamiliar product family

    - Unnecessary for requirements

    - Small operational voltage range

    PIC18LF8723-I/PT
    $16.85/each
    datasheet    		 - 128kB Memory

    - 70 IO pins

    - Allows for more redundant wiring    		 - Excessive connection points

    - Expensive

    - Overkill for requirements

    Selected Component: Option 1 - PIC18F47Q10-I/PT
    Rational: This microcontroller satisfies my group's needs for microcontroller capabilities for our project. Keeping all used microcontrollers the same aids to allow for easy integration. This option is also the most inexpensive option and allows for budget to be allocated elsewhere.

    Voltage Regulator

    Option Pros Cons

    AP63203WU-7
    $1.38/each
    datasheet    		 - Large input voltage range

    - 2A output capacity

    - Simple application circuitry    		 - Smaller operational temperature range

    - Small package

    TPS62132RGTR
    $1.71/each
    datasheet    		 - 3A output capacity

    - High switching speed

    - Lowest minimum input voltage    		 - Smaller input voltage range

    - Difficult solder pads

    - Complicated application circuitry

    LM2675MX-3.3/NOPB
    $4.36/each
    datasheet    		 - Large input voltage range

    - High maximum input voltage

    - Larger operational temperature range    		 - Limited 1A output capacity

    - Higher minimum input voltage

    - Lower switching frequency

    - Expensive

    Selected Component: Option 1 - AP63203WU-7
    Rational: This switching power supply has the desired output of 3.3 volts and has a decent current capacity. It is also in a package that is easily soldered by hand.

    Hall Effect Sensor (Wind Speed)

    Option Pros Cons

    IQS620A-0-DNR
    $0.76/each
    datasheet    		 - Inexpensive

    - I2C data interface

    - Flexible application    		 - Low operational voltage range

    - Higher current draw

    MAX9621AUB+
    $3.65/each
    datasheet    		 - Lower current draw

    - Large operational voltage range    		 - Complicated application

    - Expensive

    A1304ELHLX-T
    $0.96/each
    datasheet    		 - Simple analog output

    - Inexpensive

    - Easy implementation    		 - Cannot satisfy serial sensor requirement

    - Narrow operational voltage range

    - Higher current draw

    Selected Component: Option 1 - IQS620A-0-DNR
    Rational: This option is simple to implement, and while it has the highest current draw, its I2C communication support doesn't require additional components as an analog sensor would.

    Humidity Sensor

    Option Pros Cons

    HDC3020DEFR
    $2.55/each
    datasheet    		 - Wide operational voltage range

    - 0.5% Accuracy

    - I2C data interface

    - 4s response time    		 - Expensive

    SHT40-AD1B-R3
    $1.81/each
    datasheet    		 - 1.8% Accuracy

    - I2C data interface

    - 4s response time    		 - Narrow operational voltage range

    SHTC3-TR-10KS
    $2.06/each
    datasheet    		 - 2% Accuracy

    - I2C data interface    		 - Narrow operational voltage range

    Air Pressure Sensor

    Option Pros Cons

    DPS368XTSA1
    $2.51/each
    datasheet    		 - Supports I2C & SPI communication

    - Simpler footprint    		 - Vented guage

    - Smaller pressure range

    - Difficult soldering points

    ENS220S-BLGT
    $2.70/each
    datasheet    		 - Adequate pressure range

    - Supports I2C & SPI communication

    - Absolute pressure guage    		 - Very low operational voltage range

    - Less suited for application

    - Difficult soldering points

    LPS22HBTR
    $2.81/each
    datasheet    		 - Adequate pressure range

    - Supports I2C & SPI communication

    - Absolute pressure guage    		 - Difficult soldering points

    Selected Component: Option 3 - LPS22HBTR
    Rational: This sensor is an absolute pressure sensor that can operate with the 3.3V utilized in the subsystem. It also uses I2C to communicate data which is the chosen method for the subsystem.
